#
Chat App
This is a chat application built using React and JavaScript. The project is made by using  Firebase for authentication and secure data storage.

Features:
User authentication 
Real-time chat functionality
State management with Zustand


Setup Instructions:
   Firebase Setup:

Create a Firebase project in the Firebase Console.
Enable Firestore and Authentication services.
Set up Firestore database rules for secure data storage.
Obtain Firebase config details and include them in the project.
Install Node Modules: To run the project locally, install the necessary dependencies:

npm install

Required Node Modules:

firebase
zustand
Other basic modules for React (e.g., react-toastify for notifications)
Running the Project:

npm run dev
